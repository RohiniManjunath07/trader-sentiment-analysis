# Trader Performance vs Market Sentiment

## Objective
Analyze how market sentiment affects trader behavior and performance.

## Methodology
- Cleaned and merged sentiment + trade data
- Created metrics: PnL, win rate, trade size
- Compared Fear vs Greed behavior

## Key Insights
1. Traders perform better during Greed periods
2. Large trades increase risk during Fear
3. Frequent traders adapt better

## Strategies
- Reduce exposure during Fear
- Increase activity during Greed

## Tools Used
- Python, Pandas, Matplotlib, Seaborn
